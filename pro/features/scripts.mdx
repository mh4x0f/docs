WP3 Pro scripting workflows combine `scripts.runner`, `exec.runtime`, and session variables (`var`) for reusable automation pipelines.

### Core commands

```bash
wp3-pro > scripts.runner list
wp3-pro > scripts.runner validate <script>
wp3-pro > scripts.runner run <script>
wp3-pro > exec allowed
wp3-pro > var list
```

### Recommended flow

1. Validate script with `scripts.runner validate`.
2. Confirm runtime allowlist with `exec allowed`.
3. Capture dynamic values with `exec capture`.
4. Reuse values using `${variable.name}`.
5. Inspect runtime values with `var list`.
