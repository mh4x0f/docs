`Find My Devices` helps operators discover reachable **WiFiPumpkin3 Pro** nodes on local networks.

<Card title="device" img="/images/pro/modules/findmydevice.png">
    Find my Devices in WLAN networks
</Card>

### What it does

- Scans local ranges to identify WP3 Pro endpoints
- Speeds up onboarding in lab and distributed setups
- Reduces manual IP guessing during reconnect workflows

### Typical workflow

1. Open the discovery feature in Web UI.
2. Run a quick scan on suggested local ranges.
3. Optionally run a deeper scan if needed.
4. Connect directly to discovered nodes.

<Note>
Use this together with `Recent Devices` to speed up repeated operations.
</Note>
