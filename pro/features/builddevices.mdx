<Note title="Build your own WiFiPumpkin3 gadget">
WiFiPumpkin3 Zero is a compact, purpose-built edition of the platform designed to run on ARM64 single-board computers such as the **Raspberry Pi Zero 2 W** and **Orange Pi Zero 2W**. Despite its small footprint, it delivers a full rogue access point and field-recon toolkit in a pocket-sized package.
</Note>

![WiFiPumpkin3 Zero gadget](/images/pro/gadget3.jpeg)

### Why build a gadget?

- 📦 **Portable & discreet** — pocket-sized form factor for field operations
- 🔋 **Battery-friendly** — pair with `battery.monitor` for runtime awareness
- 🖥️ **Headless-ready** — drive everything remotely from the WebUI
- 📟 **Optional e-paper display** — status at a glance with `epaper.ui`

### Supported hardware

| Device | Architecture | Notes |
| :----- | :----------- | :---- |
| **Raspberry Pi Zero 2 W** | arm64 (aarch64) | Recommended compact build |
| **Orange Pi Zero 2W** | arm64 (aarch64) | Great alternative, Armbian-based |
| **Raspberry Pi 3 / 4 / 5** | arm64 (aarch64) | More headroom and performance |

<Tip>
Use the `arm64` release build (`wifipumpkin3-pro_linux_arm64.zip`) and follow the standard [Installation](/pro/installation) steps. An external Wi-Fi adapter with **AP mode** support is still required — see [Hardware](/pro/hardware).
</Tip>
