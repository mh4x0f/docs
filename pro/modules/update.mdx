# Update (`update`) 

The **Update Module** allows you to check for new versions of **WP3 Pro** and manage update-related operations.  
This module is essential to ensure that your installation is always up to date with the latest features and security patches.

---

### Available Commands

#### `update.check`

- **Description**: Check if a new update is available.

```bash
wp3-pro > update.check on
```
