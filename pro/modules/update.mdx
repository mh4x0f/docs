# Update (`update`)

The **Update module** handles update checks and related maintenance operations for **WiFiPumpkin3-Pro**.

## Status

- `update` (**not running**): Handles update checks and related operations.

## Commands

### `update.check`

- **Description**: Check for updates.
- **Usage**:

```bash
wp3-pro > update.check on
```

### `update.latest`

- **Description**: Update WiFiPumpkin3-Pro to the latest version.
- **Usage**:

```bash
wp3-pro > update.latest
```
